A lightweight, responsive photo blog theme featuring Polaroid image design, AJAX archive navigation, infinite scroll option, and a dark mode toggle.

== Description ==

Photodiary is a clean, lightweight WordPress theme perfect for daily photo blogs and visual journals. Features include:
* Responsive polaroid-style photo grid
* AJAX archive navigation
* Dark mode toggle
* Infinite scroll (optional)
* Gutenberg block support
* Translation ready

== Installation ==

1. Upload `photodiary` to the `/wp-content/themes/` directory.
2. Activate the theme through the 'Appearance > Themes' menu.
3. Customize settings via the WordPress Customizer.
